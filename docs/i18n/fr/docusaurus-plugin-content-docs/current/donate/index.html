Soutenir le développement de Joplin🔗

Joplin est un projet open source, et vos dons supportent le développeur. Le développement d'applications de qualité prend le plus de temps, mais il y a aussi des dépenses, comme les certificats numériques pour signer les applications, frais de magasin d'applications, hébergement ainsi que le matériel pour développer et tester une application multi-plateforme.

Surtout, votre don permettra de respecter les normes de développement actuelles. et apportera de nouvelles fonctionnalités et améliorations à l'application.
