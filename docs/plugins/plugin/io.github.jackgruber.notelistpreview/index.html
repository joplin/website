Installation

Automatic

  • Go to Tools > Options > Plugins
  • Search for Note list (Preview)
  • Click Install plugin
  • Restart Joplin to enable the plugin

Manual

  • Download the latest released JPL package (io.github.jackgruber.notelistpreview.jpl) from here
  • Close Joplin
  • Copy the downloaded JPL package in your profile plugins folder
  • Start Joplin

Usage

The new note list view can be activated under View > Note list style > Preview.

Plugin options

Settings for the plugin, accessible at Tools > Options > Note overview. Joplin must be restarted after changing settings!
