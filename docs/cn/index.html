你的笔记你在哪里都可以

通过与各种服务同步，包括Joplin Cloud、Dropbox和OneDrive，从你的电脑、手机或平板电脑访问你的笔记。该应用程序可在Windows、macOS、Linux、Android和iOS上使用。终端应用也可使用!

百分之百你的数据

该应用程序是开源的，你的笔记被保存为开放的格式，所以你将永远可以访问它们。使用端对端加密（E2EE）来保护你的笔记，确保除了你自己之外没有人可以访问它们。
