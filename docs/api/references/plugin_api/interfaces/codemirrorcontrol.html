  • completionSource: function
    • completionSource(completionSource: any): any

    • Returns a CodeMirror 6 extension that registers the given CompletionSource.

      Use this extension rather than the built-in CodeMirror autocompletion if you don't want to use langaugeData-based autocompletion.

      Using autocompletion({ override: [ ... ]}) causes errors when done by multiple plugins.
